# ความเป็นมาของ NGINX




## ความสามารถของ NGINX

- ทำเป็น web server
- ทำเป็น proxy server
- ทำเป็น load balance 

## ข้อดีของ NGINX

- 1. เร็วที่สุดและดีที่สุดสำหรับการแสดงข้อมูลไฟล์แบบคงที่

- 2. รองรับ Load Balancing

- 3. รองรับและจัดการ  concurrent connections ได้มากกว่า Apache หลายเท่า
